## Overview
This note explores key learning theories and their applications in educational technology, focusing on how these theories inform the design and implementation of digital learning environments.

## Key Learning Theories

### 1. Constructivism
**Key Theorists**: Jean Piaget, Lev Vygotsky
- **Core Principles**:
  - Active construction of knowledge through experience
  - Social interaction in learning
  - Zone of Proximal Development (ZPD)

**EdTech Applications**:
- Interactive simulations and virtual laboratories
- Collaborative learning platforms
- Project-based digital environments
- [[Digital Learning Strategy]]

### 2. Connectivism
**Key Theorist**: George Siemens (2004)
- **Core Principles**:
  - Knowledge distributed across networks
  - Learning as network navigation
  - Digital tools in knowledge creation

**EdTech Applications**:
- Online learning communities
- Personal Learning Networks (PLNs)
- Social learning platforms
- [[Learning Analytics]]

### 3. Cognitive Load Theory (CLT)
**Key Theorist**: John Sweller
- **Types of Cognitive Load**:
  1. Intrinsic Load
  2. Extraneous Load
  3. Germane Load

**EdTech Design Principles**:
- Information chunking
- Progressive disclosure
- Multimedia integration
- [[Adaptive Learning Systems]]

### 4. Social Learning Theory
**Key Theorist**: Albert Bandura
- **Core Principles**:
  - Observational learning
  - Modeling
  - Social context in learning

**EdTech Applications**:
- Peer collaboration tools
- Virtual classrooms
- Social learning features
- [[Student Engagement Guide]]
